How to Use This Application

This application provides real-time analysis of news articles.

To view the live news feed and statistics, please visit the news page:

View News Feed

For Developers

You can simulate new news articles being submitted by running the news_submitter.py script. This will send new articles to the backend, and you will see the updates in real-time on the news feed page.

The backend is built with Django and uses Server-Sent Events (SSE) to push updates to the clients.
